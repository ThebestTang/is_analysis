# 接口：setStudentSelectCourse  [返回](../README.md)
用例： [选择课程](../用例/选择课程.md)

- 功能：
    选择学生选中开课课程
    
- 权限：
    学生    
    
- API请求地址： 
    接口基本地址/v1/api/setStudentSelectCourse

- 请求方式 ：
    POST

- 请求实例：

        {
           "course_teacher_id":123
        }
        
- 请求参数说明:        

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |course_teacher_id|选中的教师开课信息id|
  
- 返回实例：

        { 
            "code":200,
            "msg":"选择成功",    
        }
 
- 返回参数说明：    
 
  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |code|返回结果状态码|
  |msg|返回结果说明信息|
