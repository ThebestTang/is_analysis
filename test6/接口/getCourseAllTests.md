# 接口：getCourseAllTests  [返回](../README.md)
用例： [查看实验](../用例/查看实验.md)

- 功能：
    查看当前课程全部实验
- 权限：
    学生，教师  
    
- API请求地址： 
    接口基本地址/v1/api/getCourseByStudent/<course_teacher_id>

- 请求方式 ：
    get

- 请求实例：
    {
    course_teacher_id:"1"
    }
        
- 请求参数说明:   
     
    |参数名称|说明|
    |:---------:|:--------------------------------------------------------|      
    |course_teacher_id|开课课程与教师关联信息id|
- 返回实例：

        { 
            "code":200,
            "msg":"选择成功",
            "data":[{
            test_id:"1",
            "test_name":"实验1",
            }]   
        }
 
- 返回参数说明：    
 
  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |code|返回结果状态码|
  |msg|返回结果说明信息|
  |data|查询到的相应信息|
